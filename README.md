# Security advisories for PolkaVM

PolkaVM doesn't publish any security advisories. The author claims it to be "unfinished" and "do not use in production", but it's already used in production in Polkadot Hub. This does sometimes give third-party teams a false sense of safety and stability guarantee.

For the benefits of third-party consumers of PolkaVM, in this repo, we publish security advisories that *should have been* published if PolkaVM is considered production software.
